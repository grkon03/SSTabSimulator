# SSTabSimulator

Semistandard Tableaux Simulator

## 解説動画について

https://drive.google.com/file/d/1-ocR1oqszFeIq7i244-j6dcB3a9mIedi/view?usp=sharing

## build

`mkdir build` is required only the first time.

```
cd src
mkdir build
cd build
cmake ..
cmake --build .
```

## execute

```
./SSTabSimulator
```
